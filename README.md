# AIMP4-playlist-to-m3u

## Usage

```sh
$ python.exe aimp2m3u.py --help
usage: aimp2m3u.py [-h] [-o OUTPUT] playlist

positional arguments:
  playlist              The AIMP4 playlist to convert

optional arguments:
  -h, --help            show this help message and exit
  -o OUTPUT, --output OUTPUT
                        The output dir

```
